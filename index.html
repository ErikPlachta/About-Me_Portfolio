About Me

In 2014, I married my best friend; the beautiful & intlligent woman, Emily Plachta.

Professionally, I've been providing IT support and service since Jan, 2008.

I cut my teeth at an IT repair center, where I provided software and hardware support to mobile and desktop hardware.

Present tense, I'm managing a IT support center that provides services to customers internationally.

Over 10 years of Management and Project Management experience.

Early in my career, I took some OOP and Programming Theory classes. I wanted to have a better undersatnding of how software works, so I could round out my skill-set as an IT professional.

It's easier to fix something once you understand how it works.

In 2017, I moved into a new career which gave me the opportunity to start to developing software professionally.

My first big project is something I call Image Manage, which is a Python program designed to automate resolutions to IT Support related issues.

See projects below for more details.

Amoung other things, my primary focus in Python has been centered around Windows, PoweShell, SQL Server, AWS, JSON, webhooks, Google Apps Script, CSV, logging, Excel and Salesforce.

My initial Web Development experience was related to Python in 2019. I needed an easy way to present data locally. I was sending JSON , with the power and flexability of a website. Prior, I was sending a webhook to a Cloud Database or an Excel file doesn't always make sense.

Started taking classes oct 2021. Seriously started indpenedent projects 2019. Always been a thing I've enjoyed.
